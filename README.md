# 🛠️ Ave1 - Web Application

Originální webová aplikace postavená od základu s důrazem na čistý kód a vlastní řešení.

## 🚀 Live Demo
Aplikace je nasazená a dostupná online:
👉 [Zobrazit živou ukázku](https://martinpiloprace-glitch.github.io/ave1/)

## 🛠️ Použité technologie
* **HTML5** (sémantická struktura rozhraní)
* **CSS3** (moderní stylování a responzivní layout)
* **JavaScript** (interaktivní logika a chování aplikace)

## ✨ Vlastnosti projektu
* Vlastní kód vytvořený bez použití předpřipravených šablon.
* Optimalizováno pro rychlé načítání a spolehlivý chod.
